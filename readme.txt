== Description ==

A simple Plugin to extend html text editor with 9 very usefull HTML 5 tags like
small
h2
h3
div
span
dfn (+title)
cite
button (+class,type)
adresse

**Get Help**

Enter a post in [the wordpress.org support forum for USEFULL HTML5 EDITOR BUTTONS Buttons](http://wordpress.org/support/plugin/usefull-html5-editor-buttons).


== Installation ==

1. Install USEFULL HTML5 EDITOR BUTTONS in your plugin folder and activate.
2. Use in HTML/Text Editor only.
